# 麻雀 7枚形ドリル

麻雀における、いわゆる「7枚形の待ち」を覚えるためのツールです。

7枚形とは何か、それを覚える意義は何かについては、多くの入門書やWebサイトで開設されていますので、そちらをご参照ください。
